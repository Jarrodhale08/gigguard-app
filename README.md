# GigGuard

Financial management for gig workers and freelancers. Track income across platforms, estimate quarterly taxes, set aside savings automatically, invoice clients.

## Features

- **Transaction Tracking**: Log income and expenses easily
- **Budget Categories**: Organize spending into categories
- **Visual Reports**: Charts and graphs of spending patterns
- **Bill Reminders**: Never miss a payment

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android
```

## Tech Stack

- **Framework**: React Native with Expo SDK 54
- **Language**: TypeScript
- **Navigation**: Expo Router
- **Styling**: NativeWind (Tailwind CSS)
- **State Management**: Zustand
- **API Client**: Axios with React Query
